# Escape the room - 3D 動作收集遊戲 ( 開發筆記 )
這是一款使用 Unity 3D 獨立開發的單人遊戲。
玩家需要在關卡中躲避敵人的追擊及避免觸發陷阱，同時蒐集指定數量的禮物以開啟傳送門通關。

## 開發資訊
* 開發環境：Unity 2023.2 ( C# )
* 開發人員：王程豐 ( Cheng-Feng Wang )
* 核心技術：uGUI、Component-Oriented Design、NavMesh AI、Prefab 等系統
* 開發日期：2024.7.10

## 遊戲機制與設計邏輯

### 1. 創新的主畫面 UI (Interactive Menu)
不同於傳統的點擊按鈕互動，初始主畫面採用了**角色導向的互動方式**。
玩家需操控角色進入特定的傳送門來進行「開始遊戲」 或 「離開遊戲」。

### 2. 核心開發系統
* **角色控制與物理判定**：基於 Unity 3D 模板進行開發，針對移動和跳躍等參數進行細節調整，並實作與敵人和互動物品 ( 禮物、回血道具 ) `isTrigger` 碰撞偵測系統。。
* **敵人 AI 系統**：利用內建功能實作**自動追蹤玩家**與**定點巡邏**等功能。
* **陷阱與機關連動**：製作壓力板觸發機制，用以控制對應陷阱的開啟與關閉。
* **視覺化 UI 系統**：使用 **uGUI** 建構即時血量與禮物計數器。

## 專案結構說明
* `Assets/Scripts`: 包含玩家控制、敵人 AI、陷阱邏輯等 C# 腳本。
* `Assets/Prefabs`: 遊戲物件的預製件管理，確保物件的實用及方便性。
* [點擊此處前往遊戲相關 C# 程式腳本。](./Assets/Script)

# **遊戲操作**
* **操作方式**：使用 `WASD` 移動、`Space` 跳躍。
* **快速體驗**：為方便測試已將各關卡通關所需禮物數量調整為 **3 個**。
* **下載試玩**：[點此前往 Google Drive 下載壓縮檔並執行 (Escape the room.exe) 進行遊玩。](https://drive.google.com/drive/folders/1YRVbZKdqxATcKZP2rdHF_j6gVRTB7wkQ?usp=drive_link)
* **注意事項**：目前版本為原型開發階段，若遊玩時出現 AI 巡邏或傷害判定等現象，還請見諒。
